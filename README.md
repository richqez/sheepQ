#CRUD 

```php
DB::getConn()->find($tablename);
```
Parameters:
* $tablename (string) -- name of table

Return type:  Arrays

```php
DB::getConn()->find_by($tablename,[$where],$isFirstOnly);
```
Parameters:
* $tablename (string) -- name of table
* $where (string) -- where clause
* $isFistOnly (boolean) -- get first row only

Return type:  Arrays
